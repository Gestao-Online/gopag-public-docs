# Política de privacidade

A GOPAG Soluções Tecnológicas em Pagamentos LTDA, inscrita no CNPJ sob o nº 58.753.292/0001-13, com sede na Avenida Fued José Sebba, 700, Quadra A5, Lote 22/23, Térreo Coworking, Jardim Goiás, CEP 74805-100, Goiânia – GO, doravante denominada “GOPAG”, disponibiliza esta Política de Privacidade para informar como coletamos, usamos, armazenamos, compartilhamos e protegemos os dados pessoais dos usuários de nossos serviços, em especial no contexto de nossa integração como parceiro da Zoop.

1. Quem é o controlador e operador dos dados? \
   \
   • A GOPAG atua como controladora dos dados pessoais coletados diretamente por seus sites e plataformas, definindo finalidades, meios e tratamento dos dados. \
   • No contexto da parceria com a Zoop, esta última poderá atuar como operadora de dados, processando os dados conforme instruções contratuais e em conformidade com a legislação aplicável, especialmente a Lei Geral de Proteção de Dados (Lei n.º 13.709/2018) (“LGPD”). \
   • A Zoop possui sua própria política de privacidade, que detalha como ela trata dados no âmbito dos serviços de pagamento. (https://www.zoop.com.br/politicas/privacidade)
2. Dados pessoais que coletamos \
   • Dados cadastrais / de identificação: nome, CPF/CNPJ, RG, documento de identidade, data de nascimento, nome da mãe, endereço, telefone, e‑mail. \
   • Dados bancários / financeiros: número de conta bancária, agência, titularidade da conta, dados de transações, comprovantes. \
   • Dados de uso / comportamento: endereço IP, logs de acesso, horário de uso, navegador, dispositivo, cookies, identificadores únicos, histórico de operações. \
   • Dados para prevenção a fraudes e conformidade: dados de crédito, informações de score, histórico de inadimplência, checagem de identidade (ex: foto, verificação documental).\
   &#x20;• Outros dados que você fornecer voluntariamente: comunicação de suporte, formulários, feedbacks, mensagens, etc.
3. Finalidades do tratamento \
   • Viabilizar o cadastro, manutenção de conta e autenticação do usuário. \
   • Realizar e processar operações de pagamento, recebíveis, repasses, conciliações etc.\
   • Atender obrigações regulatórias, fiscais e legais. \
   • Detecção, prevenção e correção de fraudes e riscos. \
   • Cumprir obrigações contratuais com você. \
   • Atendimento ao cliente, suporte, solução de conflitos. \
   • Enviar comunicações relacionadas ao serviço, notificações e atualizações. \
   • Realizar análises estatísticas, relatórios e melhorias de produto. \
   • Se autorizado, enviar comunicações comerciais, marketing e ofertas.
4. Compartilhamento de dados / destinatários \
   • Zoop, como operadora nos escopos definidos em contrato e conforme sua política de privacidade. • Instituições financeiras, bancos, adquirentes, administradoras de cartão, redes de pagamento. \
   • Prestadores de serviço de TI, segurança, hospedagem, auditoria, suporte técnico. \
   • Empresas de prevenção a fraude, bureaus de crédito, análise de risco. \
   • Autoridades públicas, judiciais ou regulatórias, quando exigido por lei ou ordem judicial. \
   • Eventuais adquirentes ou sucessores em caso de fusão, aquisição ou reorganização societária.
5. Transferência internacional de dados \
   • Se necessário, poderemos transferir seus dados para outros países. Tais transferências serão realizadas com bases legais adequadas e observando os requisitos da LGPD.
6. Prazo de retenção dos dados \
   • Manteremos seus dados pessoais pelo tempo necessário para cumprir obrigações contratuais e regulatórias, garantir a defesa em possíveis litígios e atender obrigações fiscais e contábeis. \
   • Após esse período, os dados serão excluídos ou anonimados, salvo exigência legal para sua conservação.
7. Direitos dos titulares de dados \
   • Confirmação da existência de tratamento. \
   • Acesso aos seus dados pessoais. \
   • Correção / retificação de dados incompletos, inexatos ou desatualizados. \
   • Portabilidade dos dados em formato estruturado.\
   • Eliminação dos dados tratados com base em consentimento ou que não sejam mais necessários. • Restrição de tratamento. \
   • Oposição ao tratamento de dados para finalidades específicas (ex: marketing). \
   • Revogação do consentimento. \
   • Não estar sujeito a decisões automatizadas que produzam efeitos significativos.
8. Cookies e tecnologias similares \
   • Utilizamos cookies, web beacons e tecnologias similares para identificar o usuário, melhorar a experiência, mensurar tráfego e direcionar comunicações. \
   • Você pode gerenciar ou desativar cookies via configurações do navegador, mas isso pode impactar algumas funcionalidades.
9. Segurança dos dados \
   • Adotamos medidas técnicas e organizacionais razoáveis para proteger os dados contra acesso não autorizado, divulgação indevida, alteração ou destruição. \
   • Inclui criptografia, controle de acesso, monitoramento, auditorias e treinamentos internos.
10. Encarregado de Dados / contato \
    • Você pode contatar o nosso Encarregado de Proteção de Dados (DPO) para dúvidas, solicitações ou reclamações por meio do e‑mail: contato@gopag.com.br.
11. Alterações nesta política \
    • Podemos alterar esta política a qualquer momento, notificando os usuários em caso de mudanças relevantes.
12. Disposições gerais e legais \
    • Esta Política complementa outros contratos (Termos de Uso, Contrato de Prestação de Serviços, etc.). \
    • O foro competente para controvérsias será Goiânia – GO.
